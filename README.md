# ğŸ“– TRAVEL BOOKING <a name="about-project">Blog Here</a>

**TRAVEL BOOKING** presents an innovative web application crafted using React and Redux for the front end and the dynamic Ruby on Rails framework and powered by PostgreSQL for the backend. With its intuitive interface, users can easily navigate through the app and book their next trip. The app is fully responsive and can be used on any device.
